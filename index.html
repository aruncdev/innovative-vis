Information from the image:

Each row from the below image shows a dataset of one dog that went to the space. The below image is a static but with the change of data, the visualization can be changed. The below visualization show name of each dog that went to the space, number of times it went to space, gender of the dog and fate of the dog.

Each object representation as follows:

  • Dog with tongue: Represents a female dog
  • Dog without tongue: Represents a male dog
  • Star: each star represnts one journey to orbit
  • Gold star: gold star represents the successful journey
  • Black star: black star represents the death of dog
  • Text: name of the dog

Why do I think this is a good and/or interesting visualization?

In this assignment it is mentioned that "The audience for this visualization is visitors to a musuem". So, the main focus was on visitors understanding the data in a single look and convey important information about the soviet dogs that went to the orbit. With this visualization it is easy to get all the data example first row means "a female dog named squirrel took 4 flights to space where all the 4 missions were successful".

The data is:

  • Quantitative: Number of flights taken
  • Categorical: Fate, gender of the dog
  • Other: name
  • Marks: area
  • Channels: position, shape, hue

User tasks:

  • Lookup: example: what is the gender of dog named Squirrel
  • Compare: example: compare number of flights each dog took
  • Browse: example: find the dog that took high number of flights
  • Hover: You can hover over any dog and find about the flight it took, its latin name and other facts
